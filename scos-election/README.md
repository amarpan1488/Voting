# SCOS Executive Committee Election 2026

Voting app for the SCOS board member election.

## Deploy in 5 minutes

### 1. Push to GitHub
Create a new **private** repo on GitHub and push this folder to it.

### 2. Deploy on Streamlit Community Cloud
1. Go to [share.streamlit.io](https://share.streamlit.io)
2. Sign in with your GitHub account
3. Click **"New app"**
4. Select your repo, branch `main`, and file `app.py`
5. Click **Deploy** — it'll be live in ~60 seconds

### 3. Share the URL
Send the Streamlit URL to all SCOS physicians. Each doctor:
- Enters their name
- Opens all 3 candidate statements
- Ranks candidates 1st / 2nd / 3rd
- Submits their vote

### Results
Click **"View Results (Admin)"** on the home screen and enter the password `amarpan`.

## Scoring
- 1st place vote = 3 points
- 2nd place vote = 2 points  
- 3rd place vote = 1 point

The two candidates with the highest point totals are elected.
